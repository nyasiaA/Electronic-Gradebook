## ✨ Overview

The **Electronic Gradebook** is a simple but effective application designed to help teachers organize and manage student grades digitally. It replaces traditional paper gradebooks by automating calculations and keeping all student performance data in one place.

## 🚀 Features

* 👩🏽‍🏫 Add and manage student records
* 📝 Input assignments, quizzes, and test grades
* 📊 Automatically calculate averages and final grades
* 📁 View organized student performance summaries
* ✏️ Edit or remove student entries easily
* 💾 Simple and structured data storage

## 🛠️ Technologies Used

* Python (core programming language)
* File handling (for saving and loading data)
* Optional: GUI framework such as Tkinter (if applicable)

## ▶️ How to Run the Project

1. Download or clone the repository:

   ```bash
   git clone https://github.com/yourusername/electronic-gradebook.git
   ```
2. Open the project in VS Code or any code editor
3. Run the main program:

   ```bash
   python main.py
   ```
4. Follow the prompts to start managing grades

## ⚙️ How It Works

* The user enters student names and their grades
* Data is stored and organized within the program
* The system calculates averages and final scores automatically
* Results are displayed in a clean and readable format
